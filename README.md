# PHP CRUD Application Student Management

## Description
This is a simple PHP CRUD (Create, Read, Delete) application to manage student records. It uses **PHP**, **MySQL** and **Bootstrap 5** for styling. The project allows users to insert, view, search, and delete records from the `student` database table. 

## Features
 **Insert Records:** Add new student details (Name, Password, Mobile).
 **Display/Search Records:** View all students or search by Name or Mobile.
 **Delete Records:** Remove a single student by Name/Mobile or delete all records.
 **Database:** MySQL
 **Responsive UI:** Uses Bootstrap 5 for a clean, responsive interface.

## Technologies Used
 PHP 7.x or above
 MySQL / MariaDB
 Bootstrap 5
 HTML / CSS

## Prerequisites
 XAMPP
 Web browser (Chrome, Firefox, Edge)
 Basic knowledge of PHP and MySQL
